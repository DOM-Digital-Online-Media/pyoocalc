insert_rows(self, num_rows=1, step=1, columns_to_copy=250)

source code 

Insert rows

Insert new rows at the specified position relatively to cell. After the new row insertion the content of the current rows is copied to the new rows.

Parameters:
  • num_rows (int) - Number of rows to insert. Default value=1
  • step (int) - Step of rows insertion.
  • columns_to_copy (bool) - Number of a columns to copy on insert. No copy will be performed if columns_to_copy = 0
Returns: bool
Operation result
