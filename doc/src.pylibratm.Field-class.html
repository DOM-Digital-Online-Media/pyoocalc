Insert rows

Insert new rows at the specified position relatively to cell. After the new row insertion the content of the current rows is copied to the new rows.

Parameters:
  • row (integer) - Row index relative to current Field. Default value=0.
  • step (integer) - Step of rows insertion.
  • num_columns (integer) - Number of columns to insert. Default value=1
  • offset (integer) - Rows offset. Relatively to current field. Default value=0
